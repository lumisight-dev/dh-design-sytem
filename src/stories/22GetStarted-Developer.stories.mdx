# Getting Started Guide for Developer

The **Datahouse React Design System** contains a collection of React components that can be used to build accessible and performant React applications consistent with Datahouse brand guidelines. This getting started guide provides the basic instructions to install the Datahouse React Design System components as a dependency in your application and begin using those components. The documentation for each individual component provides more detailed information including:

- An overview with contextual usage guidance and best practices
- Examples of common variations of the component such as sizes and states
- Guidance for using the components with accessiblity in mind
- A detailed description of available React props along with an interactive playground environment
